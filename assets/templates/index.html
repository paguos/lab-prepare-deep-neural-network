Visualization of a Neural Network

Lab:prepare Student Project at TU Berlin.

Project Goal

We want to visualize what happens in each layer of a (small) neural network.

What is a neural network?

The structure of Neural Networks (NNs) is inspired by simplified models of biological neurons. It consists of a web of interconnected nodes wherein each node is responsible for a simple computation. It consists of several different layers. The basic idea of NN is to simulate lots of densely interconnected brain cells inside a computer so you can get it to learn things, recognize pattersn, and make decisions in a humanlike way.

About this project

This is a student project @ TU Berlin. We are 3 students, aiming to visualize a (small) NN. We implemented a NN for image recognition (MNIST dataset), allowing a user to draw a digit in our web interface and then observing what happens in each layer of the NN.

Try it!

Simply draw a number and have a look at the output of each layer.
